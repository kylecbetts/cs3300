Problem 3:

Type Coercion is Javascript's way of converting different types in an expression so that the expression is able to be executed. Javascript does everything it can to not throw an error or exception so some interesting conversions can occur, as seen below.

Example 1: true + "1". This example returns "true1" as the boolean true gets converted to a string and then the two strings are concatenated. In Python, this would throw an exception.

Example 2: [1, 2, 3] + true. This example returns the string "1,2,3true". The array becomes one big string and true is also converted to a string before they are concatenated. In Python, this would throw an exception.
